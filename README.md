# Wugi

Wugi ("What U Gettin Into?") is a local discovery platform for restaurants, lounges, clubs, and events.

## Project structure

- `docs/` product and technical documentation
- `mobile-app/` React Native consumer app
- `dashboard/` Next.js admin and partner dashboard
- `firebase/` Firestore and Storage rules/config
- `functions/` Firebase Cloud Functions
- `scripts/` import and utility scripts
